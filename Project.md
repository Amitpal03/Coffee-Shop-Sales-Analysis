# ☕ Coffee Shop Sales Analysis (Excel Project)

## 📌 Overview
This project analyzes coffee shop sales data using Microsoft Excel to identify trends, customer behavior, and business insights. The goal is to help improve sales performance and decision-making.

## 🛠️ Tools Used
- Microsoft Excel

## 📂 Project Workflow
1. Data Cleaning (handling missing values, formatting)
2. Data Processing (using formulas & functions)
3. Data Analysis (pivot tables)
4. Data Visualization (charts & dashboard)

## 📊 Key Insights
- Identified top-selling products
- Peak sales hours and days
- Monthly sales trends
- Customer purchasing patterns
